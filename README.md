<h1 align="center">Angular 18 bacic dev environment </h1>
<h3>An Angular 18 dev environment ready to run on existing applications or to create a new application.</h3>

# ✔️ Techs:
- `Angular`
- `Docker`

# :hammer: How to run

<h4>Remove this README.md so angular generated README.md doesn't create a conflict:</h4>

<h4>Run ´docker compose -f dev.docker-compose.yml up´ to start the application</h4>

<h4>Run ´docker compose -f dev.docker-compose.yml down --rmi all --remove-orphans´ to stop the application and destroy all docker artifacts</h4>
